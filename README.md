# wheather app react + charts

### Подготовка

Скачайте репозиторий

Переименуйте .exmple.env на .env и в поле VITE_API_KEY добавьте действующий OpenWeatherMap API

npm install

### Запуск

#### Для старта development сервера:

npm run dev

#### Для сборки билда:

npm run build

#### Для старта билда:

npm run start
